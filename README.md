# 4d-polytope-visualizer

A web based canvas that visualizes the 24 cell polytope. It uses a simple perspective projection, but I'll add more projections like stereographic, orthographic, etc. 

## Optimizations

Uh... I didn't really optimize too much, just good coding habits. I just wanted to get something working. I'm sure there's a lot of room for improvement.
